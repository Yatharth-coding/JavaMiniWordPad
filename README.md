# JavaMiniWordPad

A lightweight text editor built with Java Swing that provides essential text editing features.

## Features

### File Menu
- **New**: Create a new blank document
- **Open**: Open an existing text file
- **Save**: Save the current document to a file

### Edit Menu
- **Cut**: Cut selected text
- **Copy**: Copy selected text
- **Paste**: Paste text from clipboard

### Format Menu
- **Font Size**: Change the font size of the text
- **Text Color**: Change the color of the text

### Tools Menu
- **Word Count**: Display the total number of words and characters in the document

## Requirements

- Java Runtime Environment (JRE) 8 or higher
- Java Development Kit (JDK) for compilation

## How to Compile and Run

### Compilation
```bash
javac MiniWordPad.java
```

### Running the Application
```bash
java MiniWordPad
```

## Usage

1. Launch the application by running the compiled Java class
2. A window with a text area and menu bar will appear
3. Use the menu options to:
   - Create new documents
   - Open and save files
   - Edit text (cut, copy, paste)
   - Format text (change font size and color)
   - View word and character count
